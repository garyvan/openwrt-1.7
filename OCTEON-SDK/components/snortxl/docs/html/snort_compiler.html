SnortXL Rule Compiler

1. Introduction

SnortXL rule compiler (snortrc) is an x86 based snort rule compiler which takes snort configuration file as input, extracts fast patterns from the included snort rules, optimizes them into groups and generates graph files to be provided as input to SnortXL.

Note
Rules are precompiled and ready to use for the default configuration file provided with this release. To use default configuration file without modifications, skip to the section Building and Running SnortXL. The instructions on this page are provided to enable compilation of custom configuration files.

2. Build SnortXL Rule Compiler

The following commands should be executed to generate the SnortXL rule compiler. Ensure OCTEON-SDK environment is set before proceeding.

Compile SnortXL in rule compilation mode. 

    # cd $OCTEON_ROOT/components/snortxl
    # make clean OCTEONTARGET=x86
    # make OCTEONTARGET=x86

3. Compiling Snort Rules

Specify search-method under the detection engine configuration as cav_oct_hfa to use the OCTEON® HFA engine for custom configuration files.

    config detection: search-method cav_oct_hfa

The following command should be executed on an x86 system to compile snort rules. The rule files are sourced from the snort.conf argument to the command.

    # install.x86/bin/snortrc -H -c snortRuleDir/snort.conf

The above command will compile all the snort rules and place them in the directory "snortRuleDir/graphs".

Refer to section Building and Running SnortXL to build and run SnortXL on OCTEON.
