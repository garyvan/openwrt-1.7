OCTEON® SnortXL Installation

This release contains SnortXL software sources for OCTEON® running Linux.SnortXL takes advantage of a host of OCTEON® features to accelerate Snort Intrusion Detection System. The OCTEON® HFA pattern search engine is utilized by SnortXL to offload snort pattern search processing to HFA in asynchronous mode. It also utilizes OCTEON® packet engines for flow distribution and error checking.

This release supports the device OCTEON® CN70XX EVB and SFF.

This document provides details on how to install, compile and run SnortXL on Cavium OCTEON® devices. It provides step-by-step instructions to run the SnortXL application in SEUM mode on OCTEON CN70XX EVB running Linux.

1. Introduction

SnortXL is an OCTEON® based Network intrusion detection system. Snort version 2.9.3.1 is the base open source snort software version used in SnortXL development. Various OCTEON® features have been used to enhance SnortXL software performance.

SnortXL in Simple Executive user mode can be run in Network Intrusion Detection (NIDS) for Network Traffic and File Read Mode for Pcap Files.

2. Installation

  1. Please ensure that the following prerequisites are met prior to installation. The preferred host operating systems for installation are RHEL 5.6 and RHEL 6.0. The release has the following package dependencies: 
        o OCTEON-SDK-3.1.0 (patch_release4)
    o OCTEON-LINUX-3.1.0 (patch_release4)
    o OCTEON-HFA-SDK-3.1.0-10.i386.rpm
    o libgcrypt-devel-1.4.4-7
    o libgpg-error-devel-1.4-2
    o libpcap-devel-0.9.4-15
  2. The following command can be used to install the RPM, where XX refers to the build number: 
        # rpm -ivh OCTEON-SNORTXL-LINUX-3.1.0-XX.i386.rpm
  3. OCTEON® SnortXL will be installed by default at the location /usr/local/Cavium_Networks/OCTEON-SDK/components/snortxl

3. Directory Structure
