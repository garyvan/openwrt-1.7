INTRODUCTION TO OCTEON II/III FAMILY OF HFA DEVICES

This is the documentation for Cavium Networks implementation of HFA Library.

OCTEON II/III HFA DEVICES

The Cavium Networks OCTEON II/III has pattern matching co-processor on chip. These co-processors are the most advanced hardware acceleration co-processor family for Layer 5-7 content processing.

OCTEON II & OCTEON III has pattern matching co-processor on chip. Given a set of patterns, it tries to find those patterns in the payload. The patterns must be pre-compiled as a graph and loaded into the HFA device.

OCTEON II family of HFA processors:

OCTEON® II CN60/61XX HFA DEVICE
OCTEON® II CN62/63XX HFA DEVICE
OCTEON® II CN66XX HFA DEVICE
OCTEON® II CN68XX HFA DEVICE

OCTEON III family of HFA processors:

OCTEON® III CN70/71XX HFA DEVICE

Note: HFA on 60xx is same as on 61xx, HFA on 62xx is same as 63xx and HFA on 70xx is same as 71xx.

To start pattern matching using the HFA device:

  • Compile the set of patterns to match using the HFA compiler. The resulting graph may be a cache-only graph, memory-only graph or a graph with both cache part and memory part.
  • Download the graph into the system DDR3 memory(for CN60/61/68/70/71XX) OR Download the graph into the exclusive DDR3 memory attached to the HFA device(for CN62/63/66XX)
  • Load the cache part of the graph, if any, into the cache(s)
  • Submit a pattern matching instruction

OCTEON® II CN60/61XX HFA DEVICE

The CN61xx has 1 cluster which is illustrated below: 

                            +----------------------+
                            |      cluster 0       |
                            +----------------------+
                            | 40+ KB graph cache   |
                            |      16 HTEs         |
                            +----------------------+

The Cluster consists of 16 HTEs. All 16 HTEs have access to all 40+ K graph cache.

OCTEON® II CN62/63XX HFA DEVICE

The CN63xx has 1 cluster which looks like: 

                            +----------------------+
                            |      cluster 0       |
                            +----------------------+
                            | 160+ KB graph cache  |
                            |      16 HTEs         |
                            +----------------------+
                            +----------------------+
                            | upto 4GB DDR3 memory |
                            +----------------------+

The Cluster consists of 16 HTEs and upto 4GB of DDR3 memory attached, into which graphs can be downloaded. All 16 HTEs have access to all 160+ K graph cache.

OCTEON® II CN66XX HFA DEVICE

The CN66xx also has 1 cluster and looks like: 

                            +----------------------+
                            |      cluster 0       |
                            +----------------------+
                            | 160+ KB graph cache  |
                            |     16 HTEs          |
                            +----------------------+
                            +----------------------+
                            | upto 4GB DDR3 memory |
                            +----------------------+

The Cluster consists of 16 HTEs and upto 4GB of DDR3 memory attached, into which graphs can be downloaded. All 16 HTEs have access to all 160+ K graph cache.

OCTEON® II CN68XX HFA DEVICE

The CN68xx consists of 3 clusters which are illustrated below: 

         +--------------------+-------------------+-------------------+
         |  cluster 0         |    cluster 1      |    cluster 2      |
         +--------------------+-------------------+-------------------+
         |160+ KB graph cache |160+ KB graph cache|160+ KB graph cache|
         | 16 HTEs            |   16 HTEs         |   16 HTEs         |
         +--------------------+-------------------+-------------------+

Each cluster consists of 16 HTEs. Each cluster's 16 HTEs have access to the corresponding cluster's 160+ K graph cache.

                            +----------------------+

                            +----------------------+
                            | 40+ KB graph cache   |
                            |      16 HTEs         |
                            +----------------------+

The Cluster consists of 16 HTEs. All 16 HTEs have access to all 40+ K graph cache.
