HFA PP Resource Overriders API

  • The following are optional HFA SDK behaviour customization/override routines. This is intended for situations where applications require greater control on HFA SDK's Post Processor (pp) library behaviour in the integrated application environment. Using following APIs application can override default pp resource allocators
    • hfa_dev_set_fnp_ppalloc()
    • hfa_dev_set_fnp_ppfree()
    • hfa_dev_set_fnp_ppsize()
    • hfa_dev_set_fnp_pptalloc()
    • hfa_dev_set_fnp_pptfree()
    • hfa_dev_set_fnp_pptsize()
    • hfa_dev_set_fnp_ppmatchalloc()
    • hfa_dev_set_fnp_ppmatchfree()
    • hfa_dev_set_fnp_ppmatchsize()
    • hfa_dev_set_fnp_pperror()
