OCTEON HFA SDK INSTALLATION

This HFA SDK contains software development tools and libraries to use Cavium HFA devices namely: OCTEON II CN60XX, CN61XX, CN62XX, CN63XX, CN66XX, CN68XX and OCTEON III CN70XX & CN71XX. Given a set of patterns, the HFA engines find those patterns in the input payload. The patterns must be pre-compiled using Cavium HFA RegEx Complier into a OCTEON HFA graph and loaded into memory attached to the HFA device. This document provides details on how to install, compile and run HFA SDK on Cavium HFA devices. It provides step-by-step instructions to run example applications in different run-time environments such as Linux, Simple-Exec and Simple-Exec User Mode (SE-UM).

1. Installation

Please follow the instructions in the accompanying README.txt to install the OCTEON-HFA-SDK rpm.

Note: After installation of OCTEON-HFA-SDK rpm, please clean and rebuild Linux Kernel.

2. Post Installation

The following is the directory structure after installation. These files and directories are installed by default at /usr/local/Cavium_Networks/OCTEON-SDK/components/hfa

    |-- apps

        |-- func
            This directory contains the reference programs which illustrate
            usage of different API in the HFA SDK. It also contains example
            pattern(s) and payload(s) to showcase the usage

        |-- perf
            This directory contains performance benchmarking applications.

        |-- ref-files
            This directory contain some reference files like

            |-- patterns
                Sample pattern file to generate OCTEON HFA Graph

            |-- payload
                Sample payload file and sample pattern file can be used
                together with provided reference applications and OCTEON HFA
                Walker

            |-- xml_patterns
                Sample xml pattern file which can run with hfa-se-xml-parser.c
                application.

            |-- hfa_set_irq_affinity.sh
                Sample script used by few reference KERNEL application to set
                IRQ core affinity

            |-- hfa.config
                Sample configuration file which can be provided to --config
                option of OCTEON HFA compiler 
    |-- docs
        This directory contains the documentation for HFA-toolchain and APIs.
        Please open docs/html/index.html to view them.

    |-- drv
        This directory contains source and header files for the HFA SDK
        API/module driver that interacts with OCTEON HFA engine.

        |-- src
            This directory contains all source files (*.c, *.mk)

            |-- Makefile
                This file creates HFA APIs as library module for Simple
                Executive and Simple Executive-User Mode.

        |-- include
            This directory contains all header files (*.h)

        |-- Makefile
            This file creates module driver (HFA_LIB_MODULE) that provides
            interface to run HFA applications in kernel environment.

    |-- hfa-sdk.patch
        This is a patch on the OCTEON-SDK installation. It is applied during
        rpm installation. It is provided as a reference and you can apply
        this patch manually onto another OCTEON-SDK installation other than
        in the RPM installation path.

    |-- lib
        This directory contains library binaries to run HFA applications

        |-- octeon
            This directory contains libraries for OCTEON architecture

            |-- pp
                This directory contains post-processing library for all
                environments (i.e. Simple Executive, Linux user-space (SE-UM)
                and kernel) in which HFA applications can run

            |-- hfa
               This directory contains HFA library for all three possible
               environments. HFA libraries for each operating environment
               (se/seum/kernel) are created after performing step "Compiling
               Libraries and Reference Applications"

    |-- tools
        This directory contains the HFA compiler toolchain

        |-- x86
            Tools executable on x86/i386 host machine

                a. hfac,       the HFA pattern compiler
                b. hfal,       the HFA graph linker
                c. hfav,       the HFA graph/info verifier
                d. hfaw,       the HFA core functional simulator

        |-- octeon
            Tools executable on OCTEON-Linux systems

                a. hfac,       the HFA pattern compiler for OCTEON-Linux
                b. hfal,       the HFA graph linker for OCTEON-Linux
                c. hfav,       the HFA graph/info verifier for OCTEON-Linux

    |-- utils
        This directory contains HFA utility code required for running HFA
        applications.
