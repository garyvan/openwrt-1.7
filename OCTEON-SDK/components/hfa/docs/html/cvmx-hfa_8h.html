const char* hfa_octeon_model_get_string ( uint32_t  chip_id)

Given the chip processor ID from COP0, this function returns a string representing the chip model number.

The string is of the form CNXXXXpX.X-FREQ-SUFFIX.

  • XXXX = The chip model number
  • X.X = Chip pass number
  • FREQ = Current frequency in Mhz
  • SUFFIX = NSP, EXP, SCP, SSP, or CP
